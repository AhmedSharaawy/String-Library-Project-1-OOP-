## Overview

`clsString` is a custom C++ class that groups common string operations into a single, easy-to-use class instead of relying entirely on `std::string`.

This project was built as a practice exercise to apply **Object-Oriented Programming (OOP)** concepts and implement common string algorithms manually.

---

## Main Features

- Get and set string value
- Calculate string length and word count
- Letter case conversions (upper / lower)
- Count:
  - All letters
  - Capital letters
  - Small letters
  - Vowels
- Split strings into words
- Trim spaces:
  - Left trim
  - Right trim
  - Full trim
- Join strings from arrays or vectors
- Reverse words in a string
- Replace words
- Remove punctuations

- Practice **OOP concepts** in C++
- Work with classes and member functions
- Implement common string operations manually
